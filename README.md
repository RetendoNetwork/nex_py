# NEX
- NEX Library for Wii U/3DS/Switch Games written in Python.

## Installation
`pip install nex-retendo`

## Features
- [x] HPP
- [ ] PRUDP
   - [ ] Transport UDP
   - [ ] PRUDPv0 (Friend Server & some 3DS Games)
   - [ ] PRUDPv1 (Wii U & some 3DS Games)
   - [ ] PRUDPLite (Switch)
- [x] Kerberos
- [x] RMC
