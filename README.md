# NEX
- NEX Library for Wii U/3DS/Switch Games written in Python.

## Installation
`pip install nex-retendo`

## Features
- [x] HPP
- [x] PRUDP
   - [x] Transport UDP
   - [x] PRUDPv0 (Friend Server & some 3DS Games)
   - [x] PRUDPv1 (Wii U & some 3DS Games)
   - [x] PRUDPLite (Switch)
- [x] Kerberos
- [x] RMC
